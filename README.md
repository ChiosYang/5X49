# Film Genealogy Agent

A comprehensive film analysis agent that extracts "genealogy" (cinematic DNA) from movies, identifying themes, visual styles, and philosophical underpinnings.

## 🚀 Getting Started

### Docker Deployment (Recommended)

See [README.docker.md](./README.docker.md) for detailed Docker deployment instructions.

### Local Development

**Backend:**
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
